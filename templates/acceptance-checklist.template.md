# Acceptance Checklist Template

Use this checklist to decide whether the generated report is actually acceptable.

## Data Acceptance

- [ ] Summary totals match the approved calculation rules
- [ ] Section-level counts match the canonical dataset
- [ ] Group labels and row ordering are correct

## Layout Acceptance

- [ ] Section order matches the approved form
- [ ] Table headers match the approved form
- [ ] Merged cells match the approved form
- [ ] Total rows match the approved form
- [ ] Note rows match the approved form
- [ ] Fonts, bold, italic, and alignment are acceptable

## Output Acceptance

- [ ] DOCX is editable in Microsoft Word
- [ ] PDF preserves the intended layout
- [ ] No output path silently falls back to the wrong dataset
